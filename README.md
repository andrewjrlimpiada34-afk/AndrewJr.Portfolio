# AndrewJr.Portfolio

Personal portfolio website of Andrew B. Limpiada Jr.

## Overview

This project is a React-based portfolio site that showcases:

- personal introduction
- about page
- featured project
- academic certificates
- contact and social links

## Tech Stack

- React.js
- React Bootstrap
- CSS3

## Local Development

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm start`
3. Create a production build:
   `npm run build`
